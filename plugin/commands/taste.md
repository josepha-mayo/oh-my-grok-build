# /taste — remember a preference

1. Determine whether the user is stating something they like/prefer (`like`) or something to avoid (`dislike`).
2. Run `omgb taste like <note>` or `omgb taste dislike <note>`.
3. Confirm the note was saved and mention that future prompts will include it automatically.
